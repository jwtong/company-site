# knit company-site

GatsbyJS website for Knit LLC

## Software dependencies

```sh
npm install -g gatsby-cli
```

## Project setup

```sh
npm install
```

### Compiles and hot-reloads for development

```sh
npm run develop
```

### Compiles and minifies for production

```sh
npm run build
```

### Run your tests

```sh
npm run test
```

### Lints and fixes files

```sh
npm run lint
```
